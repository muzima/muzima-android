mUzima

Lorsque vous vous connectez avec succès à l'application, vous verrez le tableau de bord mUzima. Le tableau de bord se compose de :

  1. Cohortes
  2. Client
  3. Formulaires
  4. Notifications

Cohortes:
Une cohorte est une liste de patients. Il peut s'agir d'une liste statique de patients qui ne change jamais (comme une cohorte de patients dans une étude) ou une liste dynamique de patients qui est régénérée à chaque fois (comme une cohorte de 'tous les patients de sexe masculin de plus de 65ans').

Client:
Est une liste de tous les patients qui composent les différentes cohortes téléchargées. Un nouveau patient peut également être enregistré en cliquant sur le bouton Ajouter un patient.

Formulaires:
Répertorie tous les formulaires qui ont été téléchargés, offre la possibilité de télécharger d'autres formulaires selon les besoins de l'utilisateur. Affiche les formulaires remplis et incomplets pour permettre à l'utilisateur décider de manière appropriée. Une fois que l'utilisateur a cliqué sur le menu de données complètes du formulaire, une option de téléchargement s'affiche et les formulaires complets peuvent être téléchargés.

Notification:
Répertorie toutes les communications de messages entre l'utilisateur de l'appareil et le serveur. Les messages sont regroupés comme suit:

  • Messages client - spécifiques à un client
  • Messages généraux - qui ciblent l'utilisateur.
