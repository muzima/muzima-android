Preenchimento de formulários para um paciente

Antes de preencher um formulário para um cliente, certifique-se de ter baixado o formulário correto. Se não, vá para o mUzima principal painel e clique em formulários, clique na guia 'Todos os modelos de formulário' que listará todos os formulários, escolha o formulário e clique no botão de download.

Após baixar o formulário com sucesso, vá para o painel principal do mUzima e clique em Clientes.

Siga os passos abaixo:-

  1. Selecione e clique no cliente apropriado.
  2. Na seção Formulários, você poderá ver pelo menos um ou mais formulários recomendados.
  3. Clique no formulário apropriado, ou seja, o formulário que você baixou para o telefone.
  4. O formulário será carregado com os detalhes demográficos do cliente.
  5. Preencha o formulário para o cliente.
Nota: Todos os campos marcados com um '*' vermelho são obrigatórios e DEVEM ser preenchidos Os formulários podem ser salvos como rascunho ou como completos. Cada uma das funcionalidades acima tem um botão na parte inferior do formulário
