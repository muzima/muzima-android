mUzima

Ao fazer login com sucesso no aplicativo, você verá o painel mUzima. O painel consiste em :

  1. Coortes
  2. Clientes
  3. Formulários
  4. Notificações

Coortes:
Uma coorte é uma lista de pacientes. Esta pode ser uma lista estática de pacientes que nunca mudam (como uma coorte de pacientes em um estudo) ou uma lista dinâmica de pacientes que é regenerada todas as vezes (como uma coorte de «todos os doentes do sexo masculino com mais de 65 anos»).

Clientes:
É uma lista de todos os pacientes que compõem as diferentes coortes baixadas. Um novo paciente também pode ser registrado clicando no botão adicionar paciente.

Formulários:
Lista todos os formulários que foram baixados, oferece a opção de baixar outros formulários conforme a necessidade do usuário. Mostra formulários preenchidos e incompletos para permitir que o usuário decidir adequadamente. Uma vez que o usuário clica no menu de dados de formulário completo, uma opção de upload é exibida e os formulários completos podem ser carregados.

Notificações:
Lista todas as comunicações de mensagens entre o usuário do dispositivo e o servidor. As mensagens são agrupadas como:

  • Mensagens do cliente - específicas para um cliente
  • Mensagens gerais - que visam o usuário.
